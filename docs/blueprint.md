# **App Name**: FitEvolution Hub

## Core Features:

- Responsive Navigation Bar: A fixed, responsive navigation bar including the brand logo 'Fit Evolution', links to key sections (Home, Plans, Results, Testimonials, Contact), and a prominent 'Agende Avaliação Grátis' button linking to WhatsApp.
- Motivational Hero Section: A compelling hero section featuring a full-width motivational background image, a large headline 'Transforme seu corpo em 2026 com treinos personalizados em Blumenau', a supporting subtitle, and a strong call-to-action button to WhatsApp.
- Dynamic Service Cards: A responsive section showcasing 4-5 service cards (e.g., Presencial, Online, Emagrecimento), each with an icon/emoji, title, and a brief description of the offering.
- Real Results Showcase: A 'Before/After' section displaying three pairs of placeholder images with captions, highlighting client transformations such as weight loss or muscle definition.
- Client Testimonials Display: A section featuring four testimonial cards, each with a placeholder client photo, name, a 5-star rating, and positive feedback text.
- AI-Assisted Inquiry Form: A contact form with fields for Name, WhatsApp/Telephone, Objective (using a select dropdown), and Message. This includes an AI tool that can assist by generating a relevant initial draft for the message field based on the selected objective, making it easier for users to articulate their needs, integrated with a basic EmailJS script for submissions.
- Informative Footer: A footer containing a fictional Blumenau address, WhatsApp contact, Instagram link, and a brand slogan 'Evolua com quem entende de resultados reais!', with an optional floating WhatsApp button.

## Style Guidelines:

- Color scheme is predominantly dark, providing a strong, modern, and energetic feel.
- Primary action color: A vibrant 'Verde energia' (#10B981) for key elements, symbolizing growth and vitality. This bright color contrasts sharply with the dark background for high visibility.
- Accent and call-to-action color: A bright 'Laranja motivação' (#F97316) to grab attention for critical interactions, such as 'Agende Agora' buttons, signifying enthusiasm and motivation.
- Background color: A very dark, desaturated greenish-charcoal (#1B1D16), which is derived from the primary green hue, providing a sophisticated and deep backdrop that highlights the vibrant primary and accent colors.
- Headline and body text font: 'Inter', a grotesque-style sans-serif, offering a clean, modern, and objective appearance suitable for all textual content on the landing page.
- Use clear and modern icons or emojis within the service cards and for social media links, aligning with the app's clean and energetic aesthetic.
- The layout is 100% responsive and mobile-first, featuring a fixed navbar, full-width hero image, content organized into distinct sections (cards, image pairs, testimonials), and a well-structured contact form and footer.
- Incorporate subtle animations on hover states for buttons and interactive elements, and smooth transitions between sections to enhance the user experience without being distracting.
